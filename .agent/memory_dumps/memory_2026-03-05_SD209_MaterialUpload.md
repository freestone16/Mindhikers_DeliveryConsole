## 🏗️ 架构设计要点

### 上传流程
```
用户选择 internet-clip/user-capture 条目
        ↓
点击绿勾确认 (isChecked = true)
        ↓
预览区显示"上传视频"按钮
        ↓
选择本地 MP4 文件
        ↓
POST /api/director/upload-material
        ↓
保存至: projects/${projectId}/04_Visuals/${chapterId}_rendered.mp4
        ↓
Phase 3 自动识别为"自有物料"
```

### 状态机设计
```
未选中 → 显示 "确认后上传"
已选中 + 未上传 → 显示 "上传视频" 按钮  
已选中 + 上传中 → 显示 Loading 动画
已选中 + 已上传 → 显示 "已上传" 状态 + 重新上传选项
```
