# ThumbnailMaster — 缩略图大师

> **DeliveryConsole** 二级模块  
> 负责 MindHikers 视频内容的缩略图方案、CTR 优化与认知承诺设计

---

## 这是什么

`ThumbnailMaster` 是 DeliveryConsole 的缩略图策略工作台，核心能力是在 1.5 秒内触发用户的点击冲动：

- **CTR 心理学**：基于点击心理学第一性原理设计缩略图
- **认知承诺**：视觉元素与内容承诺的高度一致性
- **A/B 测试框架**：多版本缩略图的对比验证
- **平台规范适配**：YouTube 16:9 / 其他平台规格自动适配
