# MusicDirector — 音乐总监

> **DeliveryConsole** 二级模块  
> 负责 MindHikers 视频项目的配乐方案、音乐资产管理与三模态配乐策略

---

## 这是什么

`MusicDirector` 是 DeliveryConsole 的配乐决策工作台，核心能力是为视频匹配最合适的听觉空间：

- **Artlist 实录**：人类情感驱动的真实乐器录音
- **Suno AI**：结构定制的 AI 生成音乐
- **ACE-Step 本地大模型**：高可控的本地音乐生成

覆盖从情绪分析、风格匹配、节拍同步到最终音频交付的全链路。
