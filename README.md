# ShortsMaster — 短视频大师

> **DeliveryConsole** 二级模块  
> 负责 MindHikers 视频内容的短视频化改造、病毒脚本生成与平台适配

---

## 这是什么

`ShortsMaster` 是 DeliveryConsole 的短视频策略工作台，核心能力是把长视频内容转化为具有病毒传播潜力的短视频：

- **钩子设计**：前 3 秒抓人眼球的开场
- **病毒脚本**：基于原内容提炼高传播力叙事
- **行动号召**：平台友好的 CTA 设计
- **多平台裁剪**：YouTube Shorts / TikTok / Instagram Reels 规格适配
