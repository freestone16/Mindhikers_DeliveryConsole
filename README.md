---

## 定位

`GoldenCrucibleLab` 是黄金坩埚的**实验 sandbox**。

server 端极度精简（仅 `assets.ts` 和 `chat.ts`），用于快速验证新想法、新技术或新概念，不承担正式交付职责。验证成功后，有价值的成果可迁移到 SSE 研发前线。
