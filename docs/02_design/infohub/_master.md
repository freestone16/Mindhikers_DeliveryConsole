# 📻 信息深潜 (Info Hub) 设计总纲

> **状态**: 活跃规划
> **最新修订**: 2026-03-06
> **负责人**: OldYang

## 1. 模块概述
Info Hub 是用于对外部平台（如 YouTube、X 等）进行大规模、高通量数据与信息采集和初筛的雷达站。它在整个流水线中处于最上游（P0 层），为黄金坩埚等核心大模型分析业务提供高质量的基础原料，并将成果保存到全局的 Project Store。

## 2. 架构与组件
- **平台数据接入侧 (ETL)**：包含针对 YouTube Transcript 的提取，X 平台的爬虫信息整理，并归一化输出（如使用类似 `baoyu-url-to-markdown` 等技能能力转化为 Markdown）。
- **初筛与缓存 (Processing & Store)**：对接大模型产生摘要，进行信息预归档。
- **动态导航与容器路由 (UI Navigation)**：通过 URL/Tabs 路由进行界面拆分，并动态更新 Browser Tab 以提示当前活跃的情报采集源。
