## 1. 模块概述
本模块包含支撑整套三级火箭体系、跨模块项目共享（Project Store）的数据层、配置层与隔离机制规范。基础设施的稳定性与扩展性，确保了上层各个 AI 专家进程可以在“大脑与双手分离”的安全隔离沙盒内流畅跑通。

## 2. 架构与组件
- **跨项目层级隔离 (Project Isolation)**：解决在 Antigravity 后台多开任务时的资源边界与目录一致性。
- **环境配置网关 (LLM / API Config)**：提供诸如 Volcengine，Deepseek 等多端大模型 LLM API、以及其他外部服务 API keys 的统一配置管理规范与密钥隔离（`.env`）。
- **统一服务控制 (Service Mesh)**：对终端 Node 后台与 python script 之间任务互调与状态拉取的通用基建。
