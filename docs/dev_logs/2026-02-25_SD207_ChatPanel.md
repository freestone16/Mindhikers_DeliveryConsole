## 🔧 技术要点

### 路径悬置 + 懒加载

```typescript
// 组件挂载时不加载上下文
const [contextLoaded, setContextLoaded] = useState(false);

// 用户首次发送消息时才触发
if (!contextLoaded) {
    socket.emit('chat-load-context', { expertId, projectId });
}
```

### 流式输出

```typescript
// 后端
for await (const chunk of callLLMStream(messages, provider, model)) {
    socket.emit('chat-chunk', { chunk, expertId });
}
socket.emit('chat-done', { expertId });

// 前端
socket.on('chat-chunk', ({ chunk }) => {
    setStreamingContent(prev => prev + chunk);
});
```

### 多模态适配

```typescript
const supportsImages = ['openai', 'anthropic', 'google', 'siliconflow'].includes(provider);

if (!supportsImages) {
    // 显示警告，忽略图片
    return { warning: '当前模型不支持图片输入，已忽略附件' };
}
```
