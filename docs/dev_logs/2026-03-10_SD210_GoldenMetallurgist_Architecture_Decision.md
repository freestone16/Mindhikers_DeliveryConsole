## 方法层与人格层边界

本轮明确：

- 老张 / 老卢：数字人
- `Socrates`：追问方法
- `Researcher`：真实世界资料工具
- `FactChecker`：校验工具
- `ThesisWriter`：论文成稿工具

因此：

- `Socrates` 不应在前台显名出现
- 黄金冶炼师应调用 `Socrates`，再将结果转译为老张 / 老卢的发言或中区问题卡

## 黄金冶炼师的正确定位

黄金冶炼师不应被设计为第四个常驻前台数字人，而应被写成：

- skill
- 调度器
- 导演
- 阶段控制器

它负责：

- 判断当前阶段
- 判断当前需要哪种方法
- 选择由谁对用户说
- 决定何时进入收束与论文产出

## 当前阶段输出定义

当前阶段只定义到：

- 多个平级 agent 同时争夺话语权
