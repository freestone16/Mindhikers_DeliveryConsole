### 2. 修复 scripts dropdown “像是被挡住”问题

表象：顶栏 Select script 点开后什么都没有。

根因不是 z-index，而是 scripts 请求 URL 写错：

- 错误：`/api/projects/${projectId}/scripts`
- 正确：`/api/scripts?projectId=${projectId}`

修复后：

- 顶栏脚本下拉恢复正常
- 左侧 SessionList 有真实数据
- Dock 中“当前文稿”不再是 `—`
