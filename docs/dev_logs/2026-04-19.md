### 2. 修复 scripts dropdown “像是被挡住”问题

表象：顶栏 Select script 点开后什么都没有。

根因不是 z-index，而是 scripts 请求 URL 写错：

- 错误：`/api/projects/${projectId}/scripts`
- 正确：`/api/scripts?projectId=${projectId}`

修复后：

- 顶栏脚本下拉恢复正常
- 左侧 SessionList 有真实数据
- Dock 中“当前文稿”不再是 `—`

### 3. 修复 Dock 在 260px rail 中显示拥挤

- 原先 2×2 grid 虽符合 demo 结构，但放到窄 rail 中对中文文件名不友好
- 调整为 stacked 纵向布局
- value 改为允许换行，减少强截断

### 4. 修复 Phase1 主操作区可读性

用户指出：

- `导入离线分镜 JSON` 按钮文字在深底上发灰，看不清
- 下方模板路径提示条对比度低、字号太小

调整：

- 次按钮改为深墨蓝底 + 暖米字 + 更清晰边框/阴影
- 模板提示条改为浅暖底胶囊样式，标签/路径/图标均提亮

### 5. 左侧栏节奏微调
