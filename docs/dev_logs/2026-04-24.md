- 测试仍按英文文案和旧组件 props 断言

修复策略：

- 让共享类型承认真实运行时状态，而不是在调用点继续打补丁
- 对旧数据形态做可选兼容，避免强行把新状态退回旧结构
- 更新测试到当前中文 UI 和现有组件 API

### 3. 修复 Director adapter 根因

`update_option_fields` 的问题不只是测试断言旧，而是更新语义不稳：

- `props` 浅合并会覆盖嵌套字段，例如 `textStyle`
- 修改影响画面的字段后，旧 preview 仍可能残留
- B-Roll 类型白名单落后于当前真实类型

本轮修复：

- `props` 改为深度合并
- 更新 `type/prompt/imagePrompt/template/props/quote/svgPrompt` 后显式失效旧 preview
- 补齐真实 B-Roll 类型，同时保留旧别名兼容
- 测试锁住深合并和 preview 失效行为
