## 20:23 更新

定位到本地 auth 阻塞根因：

- `.env.local` 中 `DATABASE_URL=...` 是占位符
- 旧逻辑把任意非空 `DATABASE_URL` 都视为启用 auth
- Better Auth 因此尝试连接占位符 Postgres URL，报 `Connection terminated unexpectedly`

已修复：

- `server/auth/index.ts` 新增 `resolveDatabaseUrl()`
- `DATABASE_URL` 为空或等于 `...` 时视为未配置 auth
- 本地进入无认证模式，Shell 页面可直接验收
