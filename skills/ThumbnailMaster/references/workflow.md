# 工作流与七维系统

- 内容压缩

## 内容压缩

先从长文里提炼：

- 表层话题
- 底层命题
- 目标受众
- 受众焦虑
- 核心承诺
- 反常识张力
- 情绪峰值
- 黄金句
- 可视化母题
- 禁忌俗套

压缩句格式：

`大家以为 ______ ，但这期真正要说的是 ______ ，点开后你会明白 ______。`

## 三条主路线

### 冲突驱动型

- 适用：反常识、行业变化、AI 冲击、风险预警
- 典型句式：
  - A 很强，但 B 更重要
  - 看似升级，其实在吞噬

### 价值承诺型

- 适用：方法、框架、学习、工具
- 典型句式：
  - 你会得到一个系统
  - 从混乱到有序

### 意象隐喻型

- 适用：意义、自洽、精神状态、哲思
- 典型句式：
  - 这个画面就是你现在的处境
  - 用一个物体或空间关系承载整个命题

## 七维系统

### Hook

- 先抓住什么冲突
- 例：被替代 / 被收割 / 意义贬值 / 守住金色

### Archetype

- 走哪类视觉原型
- 例：对撞 / 锚点 / 断层 / 框架 / 隐喻 / 剖面

### Layout

- 画面结构
- 例：左右对峙 / 上下反差 / 中心聚焦 / 单物体留白

### Palette

- 情绪基调
- 例：冷裂缝 / 黑金 / 米黑橙 / 冷白青

### Rendering

- 画面质感
- 例：editorial / digital / chalk / minimal cinematic

### Text

- 文字策略
- 例：无字 / 1-3 词 / 疑问句 / 反差短句

### Tension

- 张力强度
- 例：低 / 中 / 高

## 1.5 秒测试

### 0.3 秒

- 能不能从信息流里跳出来

### 0.5 秒

- 能不能快速读懂主体与问题

### 0.7 秒

- 能不能感觉“这和我有关”

如果三步里有一步失败，先回到概念卡，不要急着补 prompt。
